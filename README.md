# Simple-Ollama-LLM
1. Purpose
    • The LLM is designed to answer basic questions. These questions can be history questions, mathmematical questions and any question that has no malicious intention. The Agent can help in programming          and data analysis. The model cannot read files or images.
2. Models
    • This uses Ollama's phi3 model.

3. Requirements for ollama to work
   • One must download ollama from the official website: https://ollama.com/download
